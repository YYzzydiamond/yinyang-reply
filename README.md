# 推特阴阳回复助手 😏

一键生成贴吧老哥风格的阴阳怪气回复，专治各种不服。

## 安装方法

1. 打开 Chrome 浏览器，访问 `chrome://extensions/`
2. 开启右上角的「开发者模式」
3. 点击「加载已解压的扩展程序」
4. 选择这个文件夹 (`twitter-yinyang-extension`)
5. 插件安装完成！

## 配置 API Key

1. 点击浏览器工具栏的插件图标
2. 输入你的 DeepSeek API Key
3. 点击保存

> API Key 获取地址: https://platform.deepseek.com

## 使用方法

1. 打开推特 (twitter.com 或 x.com)
2. 找到想要回复的推文，点击回复
3. 在回复框旁边会出现「😏 阴阳」按钮
4. 点击按钮，等待 AI 生成阴阳怪气回复
5. 回复内容会自动填入输入框，点击发送即可！

## 图标文件

需要添加以下图标文件：
- `icon16.png` (16x16)
- `icon48.png` (48x48)
- `icon128.png` (128x128)

你可以用任何图片编辑工具创建，或者使用在线工具生成。
临时方案：可以先不添加图标，插件仍然可以正常工作，只是显示默认图标。

## 注意事项

- API 调用会消耗 DeepSeek 额度
- 推特页面结构可能更新，如果按钮不显示请刷新页面
- 建议适度使用，文明上网 😄
